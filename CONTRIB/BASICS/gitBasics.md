# Git Basics

[1.] Download Git onto your computer (https://git-scm.com/downloads)
[2.] Fork the repository (https://github.com/galletti94/main)
[3.] Clone your fork of the repository to your local machine
[4.] Add a remote called upstream to https://github.com/galletti94/main
[5.] Push to origin
[6.] Pull from upstream
